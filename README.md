# lscat
Uma ferramenta simples em Bash para visualizar o conteúdo de todos os arquivos de texto de um diretório de forma minimalista e organizada.

## Como usar:
Após a instalação, você pode usar o comando em qualquer diretório do seu sistema. Basta digitar: `lscat`

## Instalação:
Rode o comando abaixo e o `lscat` estará pronto para uso:

```bash
curl -sSL https://raw.githubusercontent.com/moises-h7/lscat/main/install.sh | bash
